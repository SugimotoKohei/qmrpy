# Contributor Covenant Code of Conduct

## 私たちの約束

qmrpy コミュニティでは、年齢、身体的特徴、障害、民族、性自認、経験水準、教育、社会経済的地位、国籍、外見、人種、宗教、性的指向にかかわらず、すべての参加者にとって開かれた、敬意ある環境を保つことを約束します。

## 期待される行動

- 異なる視点や経験を尊重する。
- 建設的なフィードバックを行う。
- 誤りを認め、必要に応じて修正する。
- プロジェクトとコミュニティ全体にとって最善となる行動を選ぶ。

## 許容されない行動

- 性的な表現や画像、望まれない性的関心や接近。
- 荒らし、侮辱、個人攻撃、政治的または個人的な攻撃。
- 公私を問わない嫌がらせ。
- 明示的な許可なしに他者の個人情報を公開すること。
- 専門的な場にふさわしくないその他の行為。

## 執行責任

メンテナーは、この行動規範に反するコメント、コミット、コード、Wiki 編集、issue、その他の貢献を削除、編集、拒否する責任と権利を持ちます。また、不適切、有害、攻撃的、または破壊的と判断される行動をした貢献者を、一時的または恒久的に参加停止にできます。

## 適用範囲

この行動規範は、プロジェクト空間内、および個人がプロジェクトやコミュニティを代表している場合の公的空間に適用されます。

## 報告

不適切な行動は、GitHub Issues または GitHub Security Advisories を通じて報告してください。報告内容は可能な範囲で機密に扱います。

## 出典

この行動規範は Contributor Covenant version 2.1 を基にしています。
